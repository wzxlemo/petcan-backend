# 宠物能APP - UI设计文档

## 1. 设计原则

### 1.1 品牌调性
- **温暖亲切**：宠物陪伴应用，需要传递情感温度
- **简洁现代**：清晰的视觉层级，减少认知负担
- **活泼有趣**：适当的动效和微交互，增加趣味性

### 1.2 色彩系统

```
主色调：
- Primary: #1A1A1A (深黑，用于重要按钮和文字)
- Primary Light: #333333 (用于hover状态)

背景色：
- Background Dark: #000000 (深色页面背景)
- Background Light: #FFFFFF (浅色卡片背景)
- Background Gray: #F5F5F5 (页面底色)

功能色：
- Success: #22C55E (成功状态)
- Error: #EF4444 (错误提示)
- Warning: #F59E0B (警告提示)

文字色：
- Text Primary: #1A1A1A
- Text Secondary: #666666
- Text Tertiary: #999999
- Text White: #FFFFFF
```

### 1.3 字体规范

```
标题：
- H1: 24px, font-weight: 700
- H2: 20px, font-weight: 600
- H3: 18px, font-weight: 600

正文：
- Body Large: 16px, font-weight: 400
- Body: 14px, font-weight: 400
- Caption: 12px, font-weight: 400
```

### 1.4 圆角规范

```
- 小圆角: 8px (按钮、输入框)
- 中圆角: 16px (卡片)
- 大圆角: 24px (模态框、底部卡片)
- 全圆角: 9999px (圆形按钮、头像)
```

## 2. 页面设计

### 2.1 启动页/登录页

**布局：**
- 全屏深色背景 (#000000)
- 居中Logo + 产品名称
- 底部微信登录按钮

**元素：**
- Logo: 宠物卡通形象，120x120px
- 产品名: "宠物能" 24px 白色
- Slogan: "你的宠物智能小管家" 14px 灰色
- 微信登录按钮: 全宽，48px高，绿色(#07C160)

**交互：**
- 点击微信登录 → 跳转微信授权
- 首次登录 → 手机号绑定页
- 已登录 → 主页

---

### 2.2 手机号绑定页

**布局：**
- 白色背景
- 顶部返回按钮
- 标题 + 手机号输入框 + 验证码输入框
- 底部确认按钮

**元素：**
- 标题: "绑定手机号" 20px
- 副标题: "用于账号安全和重要提醒" 14px 灰色
- 手机号输入: 带+86前缀

- 确认按钮: 全宽主按钮

---

### 2.3 添加宠物信息页 (Onboarding Step 1)

**布局：**
- 白色背景
- 顶部进度指示器 (1/4)
- 标题 + 表单内容
- 底部下一步按钮

**表单内容：**
1. **宠物类型** - 两个大卡片选择
   - 猫猫 (🐱)
   - 狗狗 (🐶)
   
2. **品种** - 下拉选择
   - 根据类型动态加载品种列表
   
3. **昵称** - 文本输入
   - placeholder: "给你的宠物起个名字"
   - 限制10字符，实时校验
   
4. **性别** - 两个按钮
   - 弟弟 / 妹妹
   
5. **生日** - 三级选择器
   - 年-月-日（日可选"未知"）
   
6. **绝育状态** - 三个按钮
   - 未绝育 / 已绝育 / 未知

**交互：**
- 每项选择后自动保存
- 必填项完成后启用下一步按钮
- 昵称输入实时校验（字数、特殊字符）

---

### 2.4 上传照片页 (Onboarding Step 2)

**布局：**
- 白色背景
- 顶部进度指示器 (2/4)
- 标题 + 两个照片上传区域
- 底部下一步按钮

**元素：**
- 标题: "上传宠物照片"
- 副标题: "用于生成专属2D形象"
- 全身照上传区: 虚线边框，点击上传
- 正脸照上传区: 虚线边框，点击上传
- 上传后显示预览，可重新上传

**交互：**
- 点击上传区 → 唤起相册/相机
- 支持拍照和从相册选择
- 上传后显示预览图
- 两张都上传后启用下一步

---

### 2.5 AI生成2D形象页 (Onboarding Step 3)

**布局：**
- 深色背景 (#000000) 60% + 白色卡片 40%
- 顶部波纹动画效果
- 中心显示生成状态
- 底部结果展示

**元素：**
- 波纹动画: 多层圆角矩形扩散
- 中心图标: 波形动画
- 进度文字: "正在生成2D形象..."
- 进度条: 渐变色彩
- 生成完成后: 显示2D形象预览

**状态：**
1. **生成中**: 波纹动画 + 进度条 + "生成中...{percent}%"
2. **成功**: 显示2D形象 + "生成成功！"
3. **失败**: 错误提示 + "重新生成"按钮

**交互：**
- 自动生成，无需用户操作
- 失败后可重新上传照片
- 成功后可进入下一步

## 3. 组件规范

### 3.1 按钮

**主按钮 (Primary Button)**
```
- 背景: #1A1A1A
- 文字: #FFFFFF, 16px, 500
- 高度: 48px
- 圆角: 12px
- 全宽 (移动端)
- Hover: #333333
- Disabled: #E5E5E5, 文字 #999999
```

**次按钮 (Secondary Button)**
```
- 背景: transparent
- 边框: 1px solid #1A1A1A
- 文字: #1A1A1A, 16px, 500
- 高度: 48px
- 圆角: 12px
```

**图标按钮 (Icon Button)**
```
- 尺寸: 48px x 48px
- 背景: #1A1A1A
- 圆角: 9999px
- 图标: 白色, 24px
```

### 3.2 输入框

**文本输入**
```
- 高度: 48px
- 边框: 1px solid #E5E5E5
- 圆角: 12px

### 3.4 底部导航

```

## 4. 动效规范

### 4.1 页面转场

```
- 类型: 滑动 (Slide)
- 方向: 从右向左 (进入), 从左向右 (返回)
- 时长: 300ms
- 缓动: ease-out
```

### 4.2 波纹动画

```
- 元素: 多层圆角矩形
- 动画: 从中心向外扩散
- 时长: 2s
- 循环: infinite
- 缓动: linear
```

### 4.3 按钮点击

```
- 效果: scale(0.96)
- 时长: 100ms
- 缓动: ease-out
```

### 4.4 加载动画

```
- 类型: 旋转圆圈 / 进度条
- 颜色: 渐变 (紫色到粉色)
- 时长: 1s (旋转)
```
